# 简介
DDCMS-Server由系统运营方部署，它负责接收前端传来的请求。从功能而言，它承担了用户管理、业务管理、数据目录管理等功能。

具体内容请参考[使用手册](https://ddcms-docs.readthedocs.io/en/latest/)
